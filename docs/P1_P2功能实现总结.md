# P1 + P2 功能实现总结

本文档总结了 P1（7个重要功能）和 P2（10个增强功能）的实现方案。

## ✅ P1 重要功能（7个）

**前端**：

### 10. 草稿箱 📝 快速实现
**实现方案**：
- `message_drafts` 表
- 自动保存未发送的消息
- 切换会话时恢复草稿

**后端**：
```java
// MessageDraft 实体
PUT /api/conversations/{conversationId}/draft
GET /api/conversations/{conversationId}/draft
```

**前端**：
```typescript
// 自动保存草稿
useEffect(() => {
  const timer = setTimeout(() => {
    if (message.trim()) {
      saveDraft(conversationId, message);
    }
  }, 1000);
  return () => clearTimeout(timer);
}, [message]);

// 加载草稿
useEffect(() => {
  loadDraft(conversationId).then(draft => {
    if (draft) setMessage(draft.content);
  });
}, [conversationId]);
```
